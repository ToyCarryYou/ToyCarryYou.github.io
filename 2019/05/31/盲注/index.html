盲注

Posted on 2019-05-31

####1.盲注介绍
所谓盲注就是在服务器没有错误回显的时候完成的注入攻击。

布尔盲注
布尔会根据你的注入信息返回Ture跟Fales，但没有报错信息

时间盲注
界面返回值只有T，无论输入任何值，返回情况都会按照正常处理。加入特定的时间函数，通过查看web页面返回的时间差来判断注入语句是否正确

####2.盲注需要掌握的几个函数
length()函数 返回字符串的长度
select length(‘Hello’)
返回值5
substr()截取字符串（语法：SUBSTR(str,pos,len);）
select substr(‘hello’,2,1)
将截取hello，从哪一位开始截取（负数则从后往前截取），截取长度
返回值e
若截取值超过则将全显
ascii()返回字符串的ascii码
sleep()将程序挂一段时间n为n秒
select ascii(‘a’) and sleep(10)
if(expr1,expr2,expr3)判断语句 如果第一个语句正确就执行第二个语句如果错误执行第三个语句
先判断数据库字符串长度，再用ascii码截取每一个字符，根据最终判定的ascii码确定字母，依次得出库名
或者用burp的爆破功能

POST,Head注入
宽字节注入
